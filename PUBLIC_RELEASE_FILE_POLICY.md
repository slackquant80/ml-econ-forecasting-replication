# Public release file policy

Included: research code required for the paper, dependency lockfiles, four FULL-run CSV result sets, paper exports, sanitized release validation, session information, deterministic checksum inventories, and the preprint PDF.

Excluded: `.RData`, `.Rhistory`, dashboard UI/deployment source, Shiny deployment metadata, PREVIEW/QUICK runs, launcher logs, internal UI audits, local-path-bearing originals, source-data CSVs, and binary RDS objects. The internal `results/ssrn/ssrn_release_validation.csv` is not modified; only a sanitized public copy is included.
